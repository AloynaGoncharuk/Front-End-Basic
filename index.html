Типи селекторів

  • універсальний селекторів - застосовується одночасно для всіх елементів сторінки, коли необхідно задати однаковий стиль для всієї сторінки
  • за назвою елемента - використовується для будь-якого елемента HTML, де визначаються правила форматування , такі як колір, фон, розмір
  • за id - визначається унікальне ім’я елемента, яке використовується для зміни його стилю
  • за класом - застосовують для різних елементів веб-сторінки. При цьому оформлення може бути частково або повністю схожим
  • за кількома класами - застосовується для для будь-якого елемента. Для елемента можна одночасно додати кілька класів, перераховуючи їх в атрибуті class через пробіл
  • за вкладеністю - за цим селектором можна встановити стиль як для окремого елемента, так і для елемента, який знаходиться всередині нього
  • за сусідством - використовується для сусідніх елементів веб-сторінки, коли ці елементи йдуть безпосередньо один за одним

Природа України – багата та різноманітна. Завдяки географічному положенню та кліматичним умовам у нас є просторі степи й густі ліси, лагідні моря й величні гори, бурхливі річки та спокійні озера, на просторах країни ростуть тисячі видів рослин та водиться величезна кількість тварин. Змінюються пори року – і кожна з них дарує нам свою особливу красу.

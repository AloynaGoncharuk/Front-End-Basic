Список команд

  • git push origin - відправити зміни в репозиторій
  • git pull - витягнути коміти в основний файл
  • git status - перевірити статус в репозиторії
  • git checkout - перейти на нову гілку
  • git commit - закомітити зміни
  • git log - дозволяє подивитись історію комітів в цілому
  • git add . - підготувати файли до коміту
